# GPIO_Embedded_Linux_examples

This repository contains simple Python examples for working with GPIO on Embedded Linux boards such as the Raspberry Pi or other ARM-based platforms.  
The code demonstrates basic digital input and output control, including buttons, LEDs, and simple sensors. Some examples also illustrate analog reading when supported by the hardware.

## Features
- Digital input (buttons, switches)
- Digital output (LEDs, relays)
- Basic sensor reading
- Optional analog input examples (hardware-dependent)
- Pure Python user-space implementation

## Requirements
- Embedded Linux board (e.g., Raspberry Pi)
- Python 3
- GPIO library such as:
  - `RPi.GPIO`
  - `gpiozero`
  - other board-specific drivers
